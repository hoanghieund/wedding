# Open Graph Images

**Purpose:** Social sharing preview images for metadata and link unfurls.

## Intended Usage

- Open Graph preview image for public pages
- Social sharing assets for links posted to messaging or social platforms
- Metadata-driven preview images

## Organization Notes

- Keep OG images sized for social preview standards
- Use this directory for metadata/social preview assets only
- Coordinate with `src/lib/metadata/` when generating page metadata
